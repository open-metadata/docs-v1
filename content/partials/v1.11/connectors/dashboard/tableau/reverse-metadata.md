## Reverse Metadata

### Description Management

Tableau supports description updates at the following levels:
- Dashboard level
- Chart level

### Owner Management

Tableau supports owner management at the following levels:
- Dashboard level

### Tag Management

Tableau supports tag management at the following levels:
- Dashboard level
- Chart level
