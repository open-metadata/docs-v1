## `metadata.sdk.data_quality.dataframes.*`

Namespace package that contains all modules required for validating DataFrames.

Assumes [`pandas`](https://pandas.pydata.org/) is installed in the python environment.

**Modules:**

- [**custom_warnings**](/sdk/python/api-reference/data-quality/dataframe-validation/custom-warnings) - Warnings specific to Dataframe validation
- [**dataframe_validator**](/sdk/python/api-reference/data-quality/dataframe-validation/dataframe-validator) â€“ Contains the main class for running tests on `DataFrame`s
- [**dataframe_validator_engine**](/sdk/python/api-reference/data-quality/dataframe-validation/dataframe-validator-engine) - Class that runs validators against `DataFrame`s
- [**validation_results**](/sdk/python/api-reference/data-quality/dataframe-validation/validation-results) - Classes containing results of test runs and failure mode options 
- [**validators**](/sdk/python/api-reference/data-quality/dataframe-validation/validators) - Registry of available validators
